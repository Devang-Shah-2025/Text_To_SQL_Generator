# Text To SQL Generator

A Streamlit web application that converts plain English questions into SQL queries using Google's Gemini AI model, and executes them against a local SQLite database.

## Overview

This project lets you query a student database using natural language. Type a question in English, and the app will:
1. Send the question to the **Gemini 2.5 Flash** model to generate the corresponding SQL query.
2. Execute that query against a local SQLite database (`student.db`).
3. Display both the generated SQL and the query results in the browser.

## Project Structure

```
Text_To_SQL_Generator/
├── sql.py            # Main Streamlit application
├── sqllite.py        # Script to create and seed the SQLite database
├── requirements.txt  # Python dependencies
├── student.db        # SQLite database (generated by sqllite.py)
└── .env              # Environment variables (not tracked — see Setup)
```

## Database Schema

The `student.db` database contains a single table:

| Column  | Type         |
|---------|--------------|
| NAME    | VARCHAR(25)  |
| CLASS   | VARCHAR(25)  |
| SECTION | VARCHAR(25)  |
| MARKS   | INT          |

## Prerequisites

- Python 3.8+
- A [Google AI Studio](https://aistudio.google.com/) API key with access to the Gemini model

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Text_To_SQL_Generator.git
cd Text_To_SQL_Generator
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure your API key

Create a `.env` file in the project root:

```env
GOOGLE_API_KEY=your_google_api_key_here
```

### 4. Initialize the database

Run the setup script once to create and seed `student.db`:

```bash
python sqllite.py
```

## Running the Application

```bash
streamlit run sql.py
```

Open the URL shown in your terminal (typically `http://localhost:8501`) in your browser.

## Usage

1. Type a natural language question in the input box.
2. Click **"Ask the question"**.
3. The generated SQL query and the database results will be displayed.

### Example Questions

| Question | Generated SQL |
|---|---|
| How many students are there? | `SELECT COUNT(*) FROM STUDENT;` |
| Show students in section A | `SELECT * FROM STUDENT WHERE SECTION='A';` |
| Who is in the Data Science class? | `SELECT NAME FROM STUDENT WHERE CLASS='Data Science';` |
| What are the marks of Devang? | `SELECT MARKS FROM STUDENT WHERE NAME='Devang';` |
